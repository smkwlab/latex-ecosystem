# ドキュメントガイド

本エコシステムのドキュメントは日本語で管理しています。

## あなたはどれ？

### 1. エコシステムの運用を新しく始める人

1. [GETTING-STARTED.md](GETTING-STARTED.md) — 運用開始までの手順ガイド

4. ルールの正典: [PR-REVIEW-GUIDELINES.md](PR-REVIEW-GUIDELINES.md)

### 3. 論文指導を受ける学生

1. 自分のリポジトリの README
2. [STUDENT-WORKFLOW.md](STUDENT-WORKFLOW.md) — 執筆プロセス全体の流れとルール

### 4. エコシステムの維持運用者

1. [../ECOSYSTEM.md](../ECOSYSTEM.md) — 全体アーキテクチャ・依存関係

3. [MANAGEMENT-WORKFLOWS.md](MANAGEMENT-WORKFLOWS.md) — 管理タスクのワークフロー
4. [SETUP-AND-RELEASE.md](SETUP-AND-RELEASE.md) — セットアップ・依存関係更新・リリース
5. [GIT-WORKFLOW.md](GIT-WORKFLOW.md) — Git 運用のベストプラクティス
